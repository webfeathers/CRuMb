CRuMb

CRuMb was created for Strike Brewing Co. in San Jose, CA, as a small application allowing them to track customers in a very simple, lightweight way. The webapp should be accessible via mobile and utilizes Twitter's Bootstrap for the presentation layer. The back end is driven by cakephp.

CakePHP is not included in this repo, but will need to be checked out and placed at the same level as the web app. For example:

  • /usr/www/CRuMb/myapp
  • /usr/www/CRuMb/cakephp

Then, your webserver needs to point the document root to:

/usr/www/CRuMb/myapp/webroot
